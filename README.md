# Employee Analytics Dashboard (Excel)

## Project Overview
This project is an interactive Excel dashboard designed to analyze employee data and provide insights into workforce composition, salary distribution, experience levels, and senior management representation.

## Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- Excel formulas

## Key KPIs
- Total Employees
- Average Salary
- Average Experience
- Percentage of Senior Management

## Dashboard Features
- Team-wise salary and experience analysis
- Gender distribution across employees
- Senior vs junior management composition
- Interactive filtering using slicers

## Dataset
The dataset contains employee-level information including gender, start date, salary, bonus percentage, senior management status and team assignment.  
This dataset is used for analytical and demonstration purposes.

## Outcome
The dashboard enables HR and business stakeholders to quickly understand workforce trends and support data-driven decision-making.
